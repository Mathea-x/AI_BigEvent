## 🛠️ 技术栈详情

### 前端技术栈
- **框架**: Vue 3 + Composition API + TypeScript
- **构建工具**: Vite
- **状态管理**: Pinia
- **路由**: Vue Router 4
- **UI 组件**: Element Plus
- **样式**: Tailwind CSS + SCSS
- **HTTP 客户端**: Axios
- **图标**: Element Plus Icons

### 开发工具
- **版本控制**: Git
- **代码规范**: ESLint + Prettier
- **API 模拟**: JSON Server
- **类型检查**: TypeScript
- **浏览器工具**: Vue Devtools

### 待集成技术
- **富文本编辑器**: Quill.js / WangEditor
- **AI 服务**: OpenAI API
- **实时通信**: WebSocket / Socket.io
- **数据可视化**: ECharts / Chart.js
- **PWA**: Workbox
- **测试**: Vitest + Vue Test Utils
